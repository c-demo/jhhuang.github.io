235.二叉搜索树的最近公共祖先

文章讲解：代码随想录
是否自己做出来通过全部用例：否

遇到的困难/犯的错误

思考和普通二叉树的区别，利用二叉搜索树的特性，简化代码逻辑。

自己写的代码

看了题解写的代码：

class Solution {
public:
    TreeNode* getcomances(TreeNode* node, TreeNode* p, TreeNode* q) {
        if (!node) return node;
        if (node->val < p->val && node->val < q->val) {
            if (TreeNode* right = getcomances(node->right, p, q)) return right;
        }
        else if (node->val > p->val && node->val > q->val) {
            if (TreeNode* left = getcomances(node->left, p, q)) return left;
        }
        else return node;
        return node;
    }
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        return getcomances(root, p, q);
    }
};

看了题解后的收获

其实这题迭代法更简单。

class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        while (root) {
            if (root->val < p->val && root->val < q->val) root = root->right;
            else if (root->val > p->val && root->val > q->val) root = root->left;
            else return root;
        }
        return root;
    }
};

两种遍历方式的递归模板：

  1. 遍历边

    if (递归函数(root->left)) return;
    if (递归函数(root->right)) return;

  2. 遍历整棵树

    left = 递归函数(root->left);
    right = 递归函数(root->right);
    left与right的逻辑处理;

701.二叉搜索树中的插入操作

文章讲解：代码随想录
是否自己做出来通过全部用例：是

遇到的困难/犯的错误

这题迭代法思路很清晰。

自己写的代码

class Solution {
public:
    TreeNode* insertIntoBST(TreeNode* root, int val) {
        TreeNode* result = root;
        while (1) {
            if (!root) return new TreeNode(val);
            if (root->val > val) {
                if (root->left == NULL) {
                    root->left = new TreeNode(val); 
                    break;
                }
                else root = root->left;
            } else {
                if (root->right == NULL) {
                    root->right = new TreeNode(val);
                    break;
                } 
                else root = root->right;
            }
        }
        return result;
    }
};

题解递归法：

class Solution {
public:
    TreeNode* insertIntoBST(TreeNode* root, int val) {
        if (root == NULL) {
            TreeNode* node = new TreeNode(val);
            return node;
        }
        if (root->val > val) root->left = insertIntoBST(root->left, val);
        if (root->val < val) root->right = insertIntoBST(root->right, val);
        return root;
    }
};

看了题解后的收获

可以用双指针法，保存上一层的节点，用上一层的节点给遍历到的NULL节点赋值。但是也可以用cur!=NULL，cur->left/cur->right == NULL来直接用父节点插入节点。

450.删除二叉搜索树中的节点

文章讲解：代码随想录
是否自己做出来通过全部用例：否

遇到的困难/犯的错误

有以下五种情况：

没找到删除节点

  • 第一种情况：没找到删除的节点，遍历到空节点直接返回了

找到删除的节点

  • 第二种情况：左右孩子都为空（叶子节点），直接删除节点， 返回NULL为根节点
  • 第三种情况：删除节点的左孩子为空，右孩子不为空，删除节点，右孩子补位，返回右孩子为根节点
  • 第四种情况：删除节点的右孩子为空，左孩子不为空，删除节点，左孩子补位，返回左孩子为根节点
  • 第五种情况：左右孩子节点都不为空，则将删除节点的左子树头结点（左孩子）放到删除节点的右子树的最左面节点的左孩子上，返回删除节点右孩子为新的根节点。

自己写的代码

这题做了一个多小时没跑通，看了题解后写的代码：

class Solution {
public:
    TreeNode* delnode(TreeNode* node, int val) {
        if (!node) return node;
        if (node->val == val) {
            if (!node->right && !node->left) return NULL;
            else if (!node->right) return node->left;
            else if (!node->left) return node->right;
            else {
                TreeNode* cur = node->right;
                while (cur->left) {
                    cur = cur->left;
                }
                cur->left = node->left;
                return node->right;
            }
        }
        if (node->val > val) node->left = delnode(node->left, val);
        if (node->val < val) node->right = delnode(node->right, val);
        return node;
    }
    TreeNode* deleteNode(TreeNode* root, int key) {
        return delnode(root, key);
    }
};

看了题解后的收获

还有一种遍历整颗树然后swap删除节点的值和它的右子树的最左节点的值的递归方法。
