39.组合总和

文章讲解：代码随想录
是否自己做出来通过全部用例：否

遇到的困难/犯的错误

这里回溯的时候startindex不需要+1。

自己写的代码

class Solution {
public:
    vector<vector<int>> result;
    vector<int> elements;
    void backtracking(vector<int>& candidates, int target, int sum, int startindex) {
        if (sum > target) return;
        if (sum == target) {
            result.push_back(elements);
            return;
        }
        for (int i = startindex; i < candidates.size(); ++i) {
            elements.push_back(candidates[i]);
            sum += candidates[i];
            backtracking(candidates, target, sum, i);
            sum -= candidates[i];
            elements.pop_back();
        }
        return;
    }
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        backtracking(candidates, target, 0, 0);
        return result;
    }
};

看了题解后的收获

本题特点：

  • 组合没有数量要求
  • 元素可无限重复选取
    本题可以不用startindex。

40.组合总和II

文章讲解：代码随想录
是否自己做出来通过全部用例：否

遇到的困难/犯的错误

去重的起始索引应该为i>startindex而不是i>0，错误写成：

if (i > 0 && candidates[i] == candidates[i-1]) continue;

会对树枝进行去重。

自己写的代码

class Solution {
public:
    vector<vector<int>> result;
    vector<int> elements;
    void backtracking(vector<int>& candidates, int target, int sum, int startindex) {
        if (sum > target) return;
        if (sum == target) {
            result.push_back(elements);
            return;
        }
        for (int i = startindex; i < candidates.size(); ++i) {
            if (i > startindex && candidates[i] == candidates[i-1]) continue;
            elements.push_back(candidates[i]);
            sum += candidates[i];
            backtracking(candidates, target, sum, i + 1);
            sum -= candidates[i];
            elements.pop_back();
        }
        return;
    }
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        sort(candidates.begin(), candidates.end());
        backtracking(candidates, target, 0, 0);
        return result;
    }
};

看了题解后的收获

理解了树枝去重和树层去重。

131.分割回文串

文章讲解：代码随想录
是否自己做出来通过全部用例：否

遇到的困难/犯的错误

没有思路，直接看题解。

自己写的代码

看了题解写的代码：

class Solution {
public:
    vector<vector<string>> result;
    vector<string> path;
    void backtracking(string s, int startindex) {
        if (startindex >= s.size()) {
            result.push_back(path);
            return;
        }
        for (int i = startindex; i < s.size(); ++i) {
            if (isbackstring(s, startindex, i)) {
                string tmp = s.substr(startindex, i - startindex + 1);
                path.push_back(tmp);
            }
            else continue;
            backtracking(s, i + 1);
            path.pop_back();
        }
    }
    bool isbackstring(string& s, int left, int right) {
        while (left < right) {
            if (s[left++] != s[right--]) return false;
        }
        return true;
    }
    vector<vector<string>> partition(string s) {
        backtracking(s, 0);
        return result;
    }
};

看了题解后的收获

本题难点：

  • 切割问题可以抽象为组合问题（套模板）
  • 如何模拟那些切割线（startindex是上一层确定的左切割线，i是本层的右切割线）
  • 切割问题中递归如何终止（切割到字符串末尾）
  • 在递归循环中如何截取子串（切割过的地方不能重复切割，故递归函数传入i+1）
  • 如何判断回文
